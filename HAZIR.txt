🎯 HEMEN BAŞLAMAK İÇİN:

1. Terminal açın
2. finans-rehberi klasörüne gidin:
   cd finans-rehberi

3. Bağımlılıkları yükleyin (ilk kez):
   npm install

4. Sunucuyu başlatın:
   npm run dev

5. Tarayıcıda açın:
   http://localhost:3000

💰 PARA KAZANMA PLANI:

1️⃣ GOOGLE ADSENSE HESABI (Ücretsiz)
   → https://www.google.com/adsense
   → Sitenizi ekle
   → Onay bekle (1-2 gün)

2️⃣ AD CODE'U PROJEDE KULLAN
   → app/layout.js → Google kod
   → app/components/AdSenseBanner.js → Slot ID'si
   → Kendi ID'niz yapıştırın

3️⃣ İçErik DOLDUR + TRAFİK ÇEKT
   → Kaliteli yazılar yaz
   → Google arama motorunda çık
   → Sosyal medyada paylaş

4️⃣ PARA KAZANMAYA BAŞLA
   → Her tıklama/gösterim = Para
   → 1000 ziyaret = ~$5-50
   → 100,000 ziyaret = ~$100-300/ay
